Template Usage Notes (For AI Agents)

When to Use Each Comment Type:

  • Strength (Green): Specific things done well - be genuine and precise
  • Teacher Comment (Yellow): Explanations, observations, teaching moments
  • Suggestion (Blue): Actionable advice - tell them HOW to improve

Inline Markup Guidelines:

  • Strikethrough + Replacement: Use for sentences that need rewriting
  • Inline Comment (Yellow highlight): Draw attention to phrases without full rewrite
  • Don't overdo it: Focus on major issues, not every tiny error

Structure Rules:

  • Always include the Overall Assessment grade box at the top
  • Break critique into sections matching the essay structure
  • Every section should have at least one comment box
  • Balance positive and constructive feedback
  • End with Final Thoughts grade box including next steps

Tone Reminders:

  • Write as an encouraging teacher, not a critic
  • Use "you" and "your" to speak directly to the student
  • Explain the "why" behind feedback
  • Frame issues as opportunities to improve
  • Celebrate genuine strengths enthusiastically
